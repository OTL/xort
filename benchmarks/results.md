## Methodology

- Both tools run under `LC_ALL=C` (byte ordering) — the conservative comparison.
- 3 timed runs, 1 warmup (page cache primed); inputs are regular files.
- Output verified **byte-identical** to GNU `sort` before timing, except the
  structured section (mlr/jq reserialize, so only timing is compared there).
- Peak RSS shows `n/a`: GNU `/usr/bin/time -v` is not available here.
